 Claude Rules and Commands

This file defines the build commands, runtime scripts, and code style conventions for the Frontend AI Engineering capstone project.

## Development & Build Commands

* **Install Dependencies:** `npm install`
* **Start Development Server:** `npm run dev`
* **Production Build:** `npm run build`
* **Linting & Code Quality:** `npm run lint`

## Code Style & Architecture

* **Tech Stack:** React, Next.js (App Router), Tailwind CSS, and TypeScript.
* **Component Structure:** Functional components with explicit TypeScript interfaces/types for props. Use hooks for state management.
* **Styling:** Strict adherence to Tailwind CSS utility classes. Avoid inline styles or custom CSS files unless absolutely necessary.
* **File Naming:** kebab-case for directories and source files (e.g., `components/custom-button.tsx`), PascalCase for component names within files.
* **Error Handling:** Use error boundaries and explicit type checks to prevent application crashes.
